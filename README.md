# 📊 Netflix Power BI Dashboard

## Overview
This project presents an interactive Power BI dashboard that analyzes Netflix movies and TV shows. The dashboard provides insights into content distribution, genres, ratings, release trends, and country-wise content availability.

## 🚀 Features
- Content distribution analysis
- Genre-wise breakdown of Movies and TV Shows
- Rating analysis
- Country-wise content insights
- Release year trends
- Interactive visualizations and filters
- Key Performance Indicators (KPIs)

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Power Query
- Data Visualization

## 📈 Dashboard Insights
The dashboard helps users:
- Understand Netflix's content library composition
- Identify popular genres and ratings
- Analyze content growth over time
- Explore country-wise content distribution
- Gain actionable business insights
